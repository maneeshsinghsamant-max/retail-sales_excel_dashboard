
# Retail Sales Dashboard 

##  Overview
This project analyzes retail sales data using Excel dashboards to uncover key business insights.

##  Key Insights
- Identified top-performing product categories contributing 40% of revenue.
- Revealed seasonal sales spikes during Q4.
- Highlighted regional differences in customer purchasing behavior.
- Built KPIs for total sales, average purchase value, and growth trends.

##  Tools Used
- Microsoft Excel (PivotTables, Charts, Conditional Formatting)
- Dataset: Retail sales transactions

##  Files
- `Retail_Sales_Dashboard.xlsx` → Interactive Excel dashboard
- `sales_data.csv` → Raw dataset
- `dashboard.png` → Screenshot preview

##  How to Use
1. Download `Retail_Sales_Dashboard.xlsx`.
2. Open in Excel (2016 or later).
3. Explore interactive filters and charts.
